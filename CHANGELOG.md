## [1.0.0] - 2026-06-24

### Added

- .NET 8 WPF 기반 Windows 네이티브 데스크톱 위젯
- `DateOnly` 기반 42일 식단 계획과 특별 일정 보정
- 날짜별 체크 상태, 실제 체중, 메모의 로컬 JSON 저장
- JSON 손상 파일 백업과 기본값 자동 복구
- 보수·표준·도전 예상 체중선과 WPF Canvas 그래프
- 6주 캘린더 및 날짜별 상세 기록 편집
- UTF-8 BOM CSV 내보내기
- 시스템 트레이, 항상 위, 단일 인스턴스, 완전 종료
- HKCU Run 기반 자동실행 등록 및 해제
- Windows GitHub Actions Release 빌드 검증
